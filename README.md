# Practice_2
Данная программа может вычислять степенные выражения, факториал натуральных чисел (1-10),
также нахождение суммы, разности, произведения произвольных чисел.
Определение времени падения тела с заданной высоты,
Нахождение величины катета в прямоугольном треугольнике по Теореме Пифагора.
Были использованы операторы условий if, else if, цикл for и типы данных int double.
